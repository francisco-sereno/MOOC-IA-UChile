📖 Instrucciones de uso

Aprende a usar el catálogo de técnicas de prompting

🎯 ¿Qué es este recurso?

Este catálogo interactivo te permite explorar y dominar 15 técnicas esenciales de prompting aplicadas al caso de Isabella Andrea, un asistente de IA generativa especializado en transición profesional.

🔍 ¿Cómo navegar?

  • Explorar técnicas: Haz clic en cualquier tarjeta para ver su descripción completa, ejemplos y aplicaciones al caso de Isabella Andrea
  • Buscar: Usa la barra de búsqueda para encontrar técnicas específicas por nombre o descripción
  • Filtrar: Usa los botones de filtro para ver técnicas por nivel (básico, intermedio, avanzado) o estado (favoritas, dominadas)

⭐ Sistema de progreso

  • Marcar como dominada: Indica que has practicado y comprendes la técnica
  • Agregar a favoritas: Guarda las técnicas que más te interesan para acceso rápido
  • Tu progreso se guarda automáticamente en tu navegador

🔄 Comparador de técnicas

Usa el comparador para analizar lado a lado dos técnicas diferentes y entender cuándo usar cada una.

♿ Accesibilidad

  • Alto contraste para facilitar la lectura
  • Navegación clara con teclado (Tab, Enter, Esc)
  • Espaciado generoso entre elementos
  • Botones grandes y visibles
